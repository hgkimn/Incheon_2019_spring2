### 주의 사항:
드론 사진측량의 결과물은 일반적으로 GSD 약3배까지의 오차를 갖는다고 알려져 있습니다.<br>
인천 청라IC에서 측량업체와 함께 확인해보니,
xy값은 약5cm안에, Z값은 약20cm안에 들어오는 것을 확인하였습니다.<br>

여기에서는 검사점을 내려 전부 확인하면 좋으나, 
비용과 활용목적을 고려해서,
국토지리정보원의 1:1000 수치지도를 사용하여 개략적인 정확도를 확인하였습니다.
